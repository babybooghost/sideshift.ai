What it does

Verify

Catch the hallucination

An independent, cross-model critic scores the claim, tags each sub-claim, and cites honestly — no invented sources.

ELI5

Break the jargon wall

Highlight a term, get a two-sentence plain-English definition in the context you're reading.

Diff

Stop re-reading whole files

Highlight code, ask for a fix, get a minimal git-style diff instead of a full reprint.

Merge / Discard

No context pollution

Close a tangent for good, or fold a short summary into the thread's context so it isn't lost.

Minimap

Never lose the thread

Minimize side-chats to colored nodes; click to jump back to what you highlighted.

Your key

Local by default

Bring your own Anthropic or OpenRouter key, stored encrypted on your machine. No server, no account.

How it works

01

Highlight

Drag a box over any text or code on screen — ⌘⇧Space, over any app.

02

Pick an action

Ask, ELI5, Verify, or Diff. Code is auto-detected and defaults to Diff.

03

Answer in place

A draggable, translucent side-chat streams the reply — the main thread never moves.
