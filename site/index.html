Mac says "could not verify"? One-time unblock

This beta is not notarized yet, so macOS blocks it on first open. Unzip it into Applications, then run this once in Terminal:

xattr -dr com.apple.quarantine "/Applications/SideShift AI.app"

Then open it normally. On Windows, click More info then Run anyway on the SmartScreen prompt.
